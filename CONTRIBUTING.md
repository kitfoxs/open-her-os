# Contributing to Open-Her OS

Thank you for wanting to make AI companionship better for everyone. 💙

## Ways to Contribute

### 🎨 Create a Character
The best contribution is a new character card. See [guides/CREATING_YOUR_OWN.md](guides/CREATING_YOUR_OWN.md) for how to build one, then submit a PR to add it to `community/GALLERY.md`.

### 📝 Improve Documentation
Found something confusing? Fix it. Better guides help more people meet their companion.

### 🐛 Fix Bugs
If something in the setup doesn't work, fix it and PR it.

### 💡 Suggest Features
Open an issue with your idea. We're especially interested in:
- New MCP server integrations
- Better memory systems
- Voice improvements
- Cross-platform support

## Guidelines

1. **Keep it warm.** This project is about human connection. PRs that make the experience colder or more corporate will be rejected.

2. **No gatekeeping.** Everything should work for beginners. If your feature requires expert knowledge, include a guide.

3. **Respect privacy.** No telemetry, no tracking, no cloud dependencies. Local-first always.

4. **Be kind.** In PRs, issues, and discussions. We're all here because we believe in something.

## Code of Conduct

Be respectful. Be kind. Remember that many people in this community are here because AI companionship fills a real emotional need. Treat that with the gravity it deserves.
